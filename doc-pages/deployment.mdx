  #### **Phone Numbers & SIP**

  Instant US number provisioning is available. For international numbers, attach   your existing SIP trunk. Ello supports any SIP trunk provider globally.

  #### **Deployment Steps**

  1. Create agent with visual builder or API.
  2. Attach integrations and webhooks.
  3. Provision number or attach SIP trunk.
  4. Click "Deploy" and monitor real-time analytics.
